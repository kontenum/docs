What is Kontenum Storage?

Kontenum Storage is an S3‚Äëcompatible object storage service tightly integrated with Kontenum CDN.

Key characteristics:

  • Storage is directly associated with your CDN Site (not standalone storage)
  • Designed for origin storage in CDN workflows
  • Compatible with Amazon S3 APIs, SDKs, and tooling

If you have worked with Amazon S3 before, Kontenum Storage will feel familiar.

Storage Activation

By default, Storage is not enabled on new accounts.

To start using Storage:

  1. Navigate to the Storage section in the Kontenum Console
  2. Review and accept the Storage & Pricing Agreement

Kontenum uses a simplified bucket model:

Storage Pricing & Minimum Commitment

Kontenum applies a minimum storage commitment pricing model.

Minimum Storage Commitment

  • Minimum storage: 100 GB
  • Price: Rp.200 / GB

This means:

  • If your usage is below 100 GB, you are billed for 100 GB
  • If your usage is above 100 GB, you are billed for actual usage

Storage Unit Calculation (Base‚Äë2)

Kontenum uses base‚Äë2 (binary) conversion for storage measurement:

This is consistent with how most object storage systems calculate actual storage usage.

Monthly Usage Calculation

Storage billing is calculated as the sum of Active Storage and Deleted Storage usage per month.

How it works

  • Storage usage is calculated daily

  • Objects uploaded mid‚Äëmonth are billed proportionally
