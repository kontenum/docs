Memulai dengan Kontenum CDN

Panduan ini akan membantu Anda mengatur CDN zone pertama dalam waktu kurang dari 5 menit.

Kontenum CDN dirancang untuk mempercepat pengiriman konten di seluruh Indonesia dengan konfigurasi minimal.

Prasyarat

Sebelum memulai, pastikan Anda memiliki:

  • Akun Kontenum yang aktif
  • Akses ke pengaturan DNS domain Anda
  • URL server origin Anda

Langkah 1: Buat CDN Zone

Masuk ke dashboard Kontenum Anda dan buat CDN zone baru.
