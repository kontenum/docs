Optimasi Gambar

Overview

Kontenum menyediakan fitur bawaan Optimasi Gambar yang tersedia tanpa biaya tambahan. Fitur ini dirancang untuk secara otomatis mengoptimalkan dan mengirimkan gambar dalam format yang paling efisien yang didukung oleh browser end user, sehingga meningkatkan performa dan mengurangi penggunaan bandwidth tanpa mengorbankan kualitas visual.

Proses optimasi gambar berjalan secara real-time (on the fly) di level CDN dan tidak memerlukan perubahan apa pun pada aplikasi atau alur upload gambar anda.

Cara Kerja

Saat fitur Optimasi Gambar diaktifkan, Kontenum CDN akan secara otomatis mendeteksi kemampuan browser dan menyajikan format gambar yang paling optimal dengan urutan prioritas berikut:

  1. AVIF
    Jika browser mendukung AVIF, Kontenum akan secara otomatis mengirimkan gambar dalam format AVIF.

  2. WebP
    Jika AVIF tidak didukung tetapi WebP didukung, gambar akan dikirimkan dalam format WebP.

  3. Format Asli yang Dioptimalkan
    Jika AVIF dan WebP tidak didukung, Kontenum akan menyajikan versi yang sudah dioptimalkan dari format gambar asli (misalnya JPEG atau PNG yang telah dioptimasi).

Proses ini sepenuhnya otomatis bagi developer maupun end user.

Menonaktifkan Optimasi Gambar

Kamu bisa menonaktifkan fitur Optimasi Gambar kapan saja melalui menu Sites Configuration di Kontenum Console.

  • Saat diaktifkan: Kontenum CDN menyajikan gambar yang sudah dioptimalkan (AVIF / WebP / format asli yang dioptimalkan).
  • Saat dinonaktifkan: Kontenum CDN menyajikan file asli (raw) persis seperti yang diunggah.

Pengaturan ini memberi kamu kontrol penuh untuk keperluan debugging, testing, atau kebutuhan kompatibilitas tertentu.

Contoh Pengurangan Ukuran File

Berikut adalah perbandingan nyata menggunakan gambar yang sama dengan kualitas visual yang sama:

  • JPEG Asli Resolusi: 4032 × 2268 pixels Ukuran file: 4.6 MB

  • Format yang Dioptimalkan oleh Kontenum CDN (Avif) Resolusi: 4032 × 2268 pixels Ukuran file: 366 KB

Hasil Optimasi

Format Ukuran File Pengurangan Ukuran
AVIF 366 KB ~92.0% lebih kecil
WebP 568 KB ~87.7% lebih kecil
Optimized JPEG 963 KB ~79.1% lebih kecil

Keuntungan

  • Biaya bandwidth dan storage lebih rendah
  • Waktu loading halaman lebih cepat
  • Kualitas visual tetap sama dengan ukuran file jauh lebih kecil
  • Kompatibilitas browser ditangani otomatis
  • Tanpa konfigurasi dan tanpa biaya tambahan

Dengan mengurangi ukuran rata-rata gambar hingga 90%, fitur Optimasi Gambar secara langsung meningkatkan efisiensi CDN sekaligus memberikan pengalaman yang lebih cepat untuk user kamu.
